# PICO ICE SPI

(Q)SPI implementation in Verilog

## Prerequisites
```
$ brew install cmake
$ brew install --cask gcc-arm-embedded
$ brew install yosys
$ brew tap ktemkin/oss-fpga
$ brew install --HEAD icestorm yosys nextpnr-ice40
$ brew install python3
```
